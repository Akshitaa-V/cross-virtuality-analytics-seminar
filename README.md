# Cross Virtuality Analytics (XVA)

This repository contains a seminar research project on Cross Virtuality Analytics (XVA), 
focusing on integrating Augmented Reality (AR), Virtual Reality (VR), and physical environments 
for immersive data visualization and collaboration.

## Key Topics
- Reality-Virtuality Continuum
- Immersive Analytics
- Visualization, Interaction, Collaboration
- Transition techniques in AR/VR
- Applications in healthcare, education, and automotive
